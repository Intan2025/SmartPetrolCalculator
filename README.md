## Project Description
Smart Petrol Cost Calculator is an Android application developed to help users calculate petrol expenses based on petrol type, petrol price, quantity of petrol purchased, and BUDI MADANI rebate eligibility.

## Features
- Select petrol type (RON95, RON97, Diesel)
- Enter petrol price per liter
- Enter quantity of petrol
- Calculate total petrol cost
- Calculate BUDI MADANI rebate savings
- Display final payable amount
- About page with developer information
- GitHub repository integration

## Technologies Used
- Android Studio
- Java
- XML
- GitHub

## Application Flow
1. User selects petrol type.
2. User enters petrol price per liter.
3. User enters quantity of petrol.
4. User selects BUDI MADANI rebate option.
5. User clicks Calculate.
6. Application displays total cost, rebate amount, savings, and final amount.
